## 함수 분석하기

```js
function* generator(i) {
  yield i;
  yield i * 2;
}

const gen = generator(10);

console.log(gen.next().value);
console.log(gen.next().value);
```

## 🔁 1. Iterator란 무엇인가?

Iterator는 `next()` 메서드를 가지고 있는 객체. `next()` 메서드는 다음 값을 반환하며 { value, done } 형식의 객체를 반환합니다.

📦 Iterator 프로토콜의 구성 요소
value: 현재 반환할 값

done: 이터레이션이 끝났는지를 나타내는 boolean

js
복사
편집
const iterator = {
next: function () {
return { value: ..., done: ... };
},
